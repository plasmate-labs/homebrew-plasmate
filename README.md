# Homebrew Plasmate

Homebrew tap for [Plasmate](https://plasmate.app) - the browser engine for AI agents.

## Install

```bash
brew tap plasmate-labs/plasmate
brew install plasmate
```

## Update

```bash
brew upgrade plasmate
```
